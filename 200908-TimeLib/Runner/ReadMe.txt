========================================================================
    응용 프로그램 : Runner 프로젝트 개요
========================================================================

응용 프로그램 마법사에서 이 Runner 응용 프로그램을 만들었습니다.  

이 파일에는 Runner 응용 프로그램을 구성하는 각 파일에 대한
요약 설명이 포함되어 있습니다.

Runner.vcproj
    응용 프로그램 마법사를 사용하여 생성한 VC++ 프로젝트의 기본 프로젝트 파일입니다. 
    파일을 생성한 Visual C++ 버전에 대한 정보와 응용 프로그램 마법사를 사용하여 선택한
    플랫폼, 구성 및 프로젝트 기능에 대한 정보가 포함되어 있습니다.

Runner.cpp
    기본 응용 프로그램 소스 파일입니다.

AssemblyInfo.cpp
	어셈블리 메타데이터를 수정하는 데 필요한 사용자 지정 특성이 포함되어 있습니다.

소스 코드 부분을 나타냅니다.
